## TangoH User Story:

1) Onboarding: Users register their identity on-chain by submitting their name and role.
2) Report: A registered user raise an Issue with a title and description.
3) Priority: Students/Excos view tickets and cast votes to support or reject the proposal.
4) Approval: An Exco approves the ticket by setting the fundraising target amount, start time, and end time.
5) Fundraising: Users can contribute to the ticket for support.
6) Conclusion: The fundraising phase is officially concluded once the target amount is reached.
7) Disbursement: The raised funds are withdrawn to a specified recipient address to start working on the Project.
8) Completion: The ticket is marked as "complete" to signal the end of the project lifecycle.
9) Acknowledgement: The ticket creator formally acknowledges the completion to validate the Issues has been ressolved.
